# Known Limitations

- This candidate composes one normalized conversation and one authority-intelligence result at a time.
- Spine selection is deterministic and classification-driven; it is not a semantic project model.
- Supersession, revocation, amendment, and dependency graphs are not resolved.
- The composer preserves upstream authority judgments but does not perform review adjudication.
- Markdown output is intentionally plain and is not a publication-layout engine.
- The ZIP package is local and does not provide hosted collaboration or cloud custody.
