# Relationship to Project Foreman

Project Foreman is the public product. This repository is its downstream artifact-composition component.

Project Foreman presents the recovered workspace. This package creates the underlying traceable Project Spine, Authority Ledger, Continuation Brief, Source Trace Index, manifest, checksums, and portable ZIP package from approved upstream evidence.

The component remains useful on its own, but its public purpose is to make Project Foreman's outputs reproducible, inspectable, and transferable.
